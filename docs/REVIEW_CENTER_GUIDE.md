# AI 审核中心使用指南

## 功能概述

AI 审核中心是一个独立的视频质量审核管理模块，提供：
- 📊 视频质量评分（0-100分）
- 🎯 四维度评分：内容质量、字幕准确性、标题吸引力、剪辑质量
- 💡 智能修复建议
- 📝 审核历史记录
- 🏷️ 自动文件重命名（添加分数标记）

## 访问入口

在顶部导航栏点击 **🎯 AI 审核中心**

## 主要功能

### 1. 视频列表与筛选

**统计面板**
- 待审核：未审核的视频数量
- 已通过：评分达标的视频数量
- 未通过：评分不达标的视频数量
- 已跳过：手动跳过审核的视频数量

**筛选器**
- 全部：显示所有视频
- 待审核：只显示未审核的视频
- 已通过：只显示通过审核的视频
- 未通过：只显示未通过审核的视频
- 已跳过：只显示跳过审核的视频

### 2. 视频审核

**开始审核**
1. 在视频卡片上点击 **"开始审核"** 按钮
2. 系统调用 Gemini 2.5 Flash 分析视频
3. 显示审核结果：
   - 综合得分（0-100分）
   - 四个维度的详细评分
   - 修复建议（如果未通过）
4. **文件自动重命名**：审核完成后，文件名会自动添加分数标记
   - 例如：`video.mp4` → `video_[85分].mp4`

**审核状态**
- ⚠️ 待审核：黄色标记
- ✓ 通过：绿色标记 + 分数
- ✗ 未通过：红色标记 + 分数
- 已跳过：橙色标记

### 3. 评分详情

审核完成后显示四个维度的评分：
- **内容质量**（默认权重 30%）：视频内容的质量和吸引力
- **字幕准确性**（默认权重 25%）：字幕的准确性和可读性
- **标题吸引力**（默认权重 20%）：标题的吸引力和相关性
- **剪辑质量**（默认权重 25%）：视频剪辑的流畅度和专业性

### 4. 修复建议

如果视频未通过审核，系统会提供具体的修复建议：
- 按类别分类（内容、字幕、标题、剪辑）
- 指出具体问题
- 提供改进方向

### 5. 其他操作

**重新审核**
- 对于未通过的视频，可以点击 **"重新审核"** 按钮
- 修改视频后重新评分

**跳过审核**
- 点击 **"跳过审核"** 按钮
- 标记为已跳过，不影响发布

**删除记录**
- 点击 **"删除记录"** 按钮
- 删除审核记录，视频恢复为待审核状态

## 审核设置

点击右上角 **"审核设置"** 按钮，可以配置：

### 基础设置
- **启用自动审核**：视频生成后自动触发审核
- **最低通过分数**：设置通过审核的最低分数（0-100）

### 权重配置
调整四个维度的权重（总和应为 100%）：
- 内容质量权重
- 字幕准确性权重
- 标题吸引力权重
- 剪辑质量权重

## 文件命名规则

审核完成后，文件会自动重命名：

**命名格式**
```
原文件名_[分数]分.扩展名
```

**示例**
- `my_video.mp4` → `my_video_[85分].mp4`
- `test_video_[70分].mp4` → `test_video_[92分].mp4`（重新审核后更新分数）

**优点**
- 在发布中心可以直接看到视频分数
- 方便筛选高质量视频
- 审核历史一目了然

## 与发布中心的集成

审核完成后：
1. 在发布中心的素材列表中，文件名会显示分数
2. 可以根据分数快速筛选高质量视频
3. 创建发布任务时，系统会检查审核状态
4. 未通过审核的视频会提示查看建议或跳过

## 环境变量配置

在 `.env` 文件中配置：

```env
# 启用审核
AI_REVIEW_ENABLED=true

# 最低通过分数
AI_REVIEW_MIN_PASS_SCORE=70

# 使用的模型
AI_REVIEW_GEMINI_MODEL=gemini-2.5-flash

# 审核超时时间（秒）
AI_REVIEW_TIMEOUT=120

# 需要手动确认
AI_REVIEW_REQUIRE_MANUAL_CONFIRM=true

# 错误时自动跳过
AI_REVIEW_AUTO_SKIP_ON_ERROR=false
```

## API 接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/review/config` | GET | 获取审核配置 |
| `/api/review/config` | POST | 更新审核配置 |
| `/api/review/video` | POST | 执行视频审核（支持 `renameFile` 参数） |
| `/api/review/skip` | POST | 手动跳过审核 |
| `/api/review/history` | GET | 获取审核历史 |
| `/api/review/:reviewId` | GET | 获取单个审核记录 |
| `/api/review/:reviewId` | DELETE | 删除审核记录 |

## 工作流程

### 手动审核流程
```
1. 进入 AI 审核中心
2. 查看待审核视频列表
3. 点击"开始审核"
4. 等待 AI 分析（通常 10-30 秒）
5. 查看评分和建议
6. 文件自动重命名（添加分数）
7. 决定：
   - 通过 → 前往发布中心创建任务
   - 未通过 → 根据建议修改后重新审核
   - 跳过 → 直接发布
```

### 自动审核流程
```
1. 视频生成完成
2. 自动触发 AI 审核
3. 审核结果保存到元数据
4. 文件自动重命名
5. 在发布中心查看结果
```

## 注意事项

1. **审核需要 Gemini API**：确保配置了有效的 `GEMINI_API_KEY`
2. **审核时间**：根据视频长度，审核可能需要 10-60 秒
3. **文件重命名**：审核完成后文件会自动重命名，原文件名会被保留在元数据中
4. **重复审核**：重新审核会更新文件名中的分数
5. **并发限制**：建议不要同时审核过多视频，避免 API 限流

## 故障排查

### 审核失败
- 检查 `GEMINI_API_KEY` 是否配置正确
- 检查网络连接
- 查看服务器日志：`data/logs/server.log`

### 文件未重命名
- 检查文件权限
- 确保文件未被其他程序占用
- 查看服务器日志中的重命名错误信息

### 评分不准确
- 调整权重配置
- 检查视频元数据是否完整
- 确保视频有字幕和标题信息
