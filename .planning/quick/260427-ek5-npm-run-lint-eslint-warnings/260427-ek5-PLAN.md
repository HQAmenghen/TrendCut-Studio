# Quick Task Plan: 清理 npm run lint 的 ESLint warnings

## Goal

把 `npm run lint` 的历史 warning 清到 0，同时避免业务行为变更。

## Scope

- 检查 1142 个 warning 的类型和集中位置。
- 对可自动修复的 ESLint 风格 warning 使用 `npm run lint:fix`。
- 手工处理剩余 `no-unused-vars` warning。
- 通过 lint 和测试验证清理结果。

## Verification

- `npm run lint`
- `npm test -- --runInBand`
