# Quick Task: Check, Package, and Push Latest Changes

## Goal

Review the current working tree, keep runtime artifacts out of the shipment, run the repository verification gates, commit source/planning changes, and push `main` to `origin`.

## Steps

- Inspect unstaged and untracked changes.
- Stage only source, tests, prompts, and planning artifacts that belong in git.
- Run JavaScript tests, Python tests, frontend build, lint, and production audit.
- Commit the staged shipment with a clear message.
- Push the current branch to `origin`.
